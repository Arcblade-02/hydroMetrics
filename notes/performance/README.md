# Performance Validation Suite

This directory stores reproducible performance validation logs generated by:

```r
Rscript tools/run_performance_suite.R
```

Generated artifacts:

- `microbenchmark_results.txt`
- `large_vector_test.txt`
- `matrix_scaling_test.txt`
- `memory_stability_test.txt`
- `registry_profile.out`
- `registry_profile_summary.txt`
- `edge_case_results.txt`
- `load_time.txt`
- `performance_summary.csv`
- `session_info.txt`
